# Brick explainer

An interactive 3D explainer for brick bonding patterns, with smooth animated transitions between steps.

## Stack

- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [React Three Fiber](https://docs.pmnd.rs/react-three-fiber) / [Three.js](https://threejs.org/) for 3D rendering
- [Drei](https://github.com/pmndrs/drei) for camera helpers
- [Vite](https://vitejs.dev/) for bundling

## Getting started

```bash
npm install
npm run dev
```

Then open `http://localhost:5173`.
